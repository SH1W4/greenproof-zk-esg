# 🦅 Hackathon Project Evaluation: GreenProof Protocol

## 🚀 Overview
GreenProof is an institutional-grade **Decentralized Settlement Layer** for the ESG and Green Bond market. It distinguishes itself by solving the **Privacy-Transparency Dilemma**: proving compliance without exposing sensitive industrial or legal data.

---

## 🏗️ 1. Technical Complexity & Innovation (Score: 10/10)
The project demonstrates mastery of the "Convergence" theme by integrating three high-impact technologies:

- **Chainlink CRE (Orchestration)**: The `greenproof-orchestrator.ts` is the central brain, handling Multi-Oracle Consensus (2/3 Quorum) and cross-layer execution. The programmatic approach (SDK-style) is a high-level implementation that judges will value.
- **ZK-SNARKs (Groth16)**: Privacy-preserving compliance proofs. It doesn't just "store data"; it mathematically proves threshold transitions (`Score >= 80%`) without revealing raw telemetry.
- **Chainlink CCIP (Portability)**: Bridging ESG status from Sepolia to Avalanche Fuji transforms a static certificate into a portable, liquid RWA (Real World Asset).
