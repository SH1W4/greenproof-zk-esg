# Contributing to GreenProof Protocol

First off, thank you for considering contributing to the GreenProof Protocol! As a sovereign settlement layer for ESG reality, we value precision, security, and decentralized decentralization.

## Institutional Standards
All contributions must align with our core HAAS (Hierarchical Autonomous Agent Swarm) architecture.

### How Can I Contribute?
1. **Physical Nucleus (IoT)**: Adding support for new environmental sensor gateways.
2. **Juridical Core (NLP)**: Improving the NLP-based legal audit engines.
3. **Ethical Engine**: Proposing new governance benchmarks for the 2026-2030 cycle.
4. **ZK Membrane**: Optimizing Circom circuits for faster threshold generation.

### Submission Process
- Fork the repository.
- Synchronize your workspace via `./scripts/sovereign-sync.sh`.
- Create a feature branch (e.g., `feat/nucleus-air-quality`).
- Submit a PR with a detailed technical rationale.

## Symbeon Labs Stewardship
The GreenProof Foundation remains the primary custodian of the protocol's IP. By contributing, you agree that your code will be released under the project's MIT License.
